HTML (Hypertext Markup Language, ou em português Linguagem de Marcação de Hipertexto) é o código usado para construir páginas na web e seus conteúdos. Por exemplo, o conteúdo pode ser estruturado em parágrafos, listas, marcadores ou usando imagens e tabelas.

Então, o que é HTML?

Diferente do que muitos acreditam, o HTML não é uma linguagem de programação; ela é uma linguagem de marcação que define a estrutura do conteúdo. O HTML consiste em uma série de elementos que você pode usar para separar ou agrupar diferentes partes do conteúdo para que ele apareça ou funcione de uma determinada maneira. Tags adicionais podem transformar texto ou imagens em hiperlinks, colocar texto em itálico, aumentar ou diminuir as fontes e muito mais.

Estrutura Básica

Uma página HTML básica vai sempre conter três tags básicas. Mas primeiro o que são tags? Tag é o nome de um elemento envolvido por sinais de maior e menor, exemplo <html> sinalizando ao navegador como um determinado trecho da página deve ser interpretada, e por isso na maioria, mas não em todas as tags será necessário uma tag de fechamento, que nada mais é do que a mesma tag de abertura com a adição de uma barra (/), ou seja, para fechar a tag <html> devemos seguir da seguinte forma , exemplo <html>.

A seguir, já dentro das tags <html> teremos mais duas tags principais. A tag <head> que é um cabeçalho e a tag <body> que é o corpo da página, e é aqui que deve ficar o elementos que serão exibido na tela para o usuário.

A tag <head> interpretada pelo não navegador antes da tag <body>. Uma tag que usada dentro da tag <head> e a tag <title> que serve para definir o título da página. Veja o exemplo abaixo:


<html>
    <head>
        <title>Título da Página</title>    
    </head>
    
    <body>
        Conteúdo da Página.
    </body>                
</html>

Outras Tags importantes para formatação do Texto

A tag <i> transforma o que estiver dentro dela em itálico, que são as letras levemente inclinadas para frente. Exemplo:

  • <i>Tags importantes para formatação do Texto</i>
Resultado da tag <i>
  • Tags importantes para formatação do Texto

A tag <strong> transforma o que estiver dentro dela em Negrito, que são letras levemente mais grossas e mais escuras. Exemplo:

  • <strong>Tags importantes para formatação do Texto</strong>
Resultado da tag <strong>

As Tags também podem ser combinadas

Caso eu queira que um determinado texto fique itálico e negrito ao mesmo tempo posso usar as tags da seguinte forma:

  • <strong><i>Tags importantes para formatação do Texto</i></strong>

Atributos das Tag

O atributos nada mais são, que propriedades que uma tag pode ter. Algumas são mais genéricas, que se aplicam a maioria das tags, já outras são mais específicas de uma determinada tag. Vejamos alguns atributos mais genéricos.

Em praticamente todas as tags podemos atribuir as propriedades Em praticamente todas as tags podemos atribuir as propriedades style, class e id. Onde style é uma propriedade de estilização capaz de mudar por exemplo a cor de um texto, porém ela só altera na tag em que ela estiver inserida.
